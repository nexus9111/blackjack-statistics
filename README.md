## Environment Variables

To run this project, you will need to add the following environment variables at the top of index.js

`NUMBER_OF_GAMES`

`BET_AMOUNT`

`STARTING_WALLET`

`TARGET_WALLET`

You can also edit the `playerTurn` funcion as you need
